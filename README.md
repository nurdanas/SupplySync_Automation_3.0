🚚 SupplySync Automation Project 3.0

📝 Project Overview

SupplySync is a web-based logistics and operations management platform. This project is a comprehensive automated testing suite designed to ensure the stability of core features such as authentication, branch management, master records, and order processing.


🛠 Tech Stack
Language: Java 17



Framework: TestNG (Testing Framework)

Library: Selenium WebDriver (Browser Automation)

Design Pattern: Page Object Model (POM)

Build Tool: Maven

Utilities: Reusable Driver, Config Reader, and Smart Waits

🚀 Key Features Demonstrated
This project showcases advanced TestNG features for professional automation:

XML Suites: Organized execution through smoke.xml and regression.xml.

Grouping: Targeted test runs using @Test(groups = {"smoke", "regression"}).

Data Driven: Efficient testing with @DataProvider.

Execution Control: Implementation of priority and dependsOnMethods.

Parallel Execution: Optimized run time via testng.xml parallel settings.

Smart Utilities: Centralized Driver class and WaitUtils to avoid hardcoded Thread.sleep().
